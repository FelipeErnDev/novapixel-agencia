# NovaPixel Agencia

Site institucional da NovaPixel Agencia, desenvolvido com React + Vite.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build para produção (Hostinger)

```bash
npm run build
```

Os arquivos estáticos ficam em `dist/`. Faça upload do conteúdo dessa pasta na Hostinger (hospedagem estática ou public_html).

## Preview local do build

```bash
npm run preview
```
